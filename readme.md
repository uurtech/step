# step

Easy Alias Management tool

**Arguments**

```shell
    step {ALIAS} {COMMAND}
```

**init**

```
step init
```

**Store**

to create a new alias

```shell
step store customer_1 -c "ssh user@IP"
```

**load**

if you have issues on source you can load the source file by calling load method

```
step load
```

**Example Call**

```shell
    customer_1
```

**Need a hint ?**

```shell
    step -h
```
