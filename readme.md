# step

Easy SSH management tool with encryption, you can think of alias for ssh connections with encryption

**Arguments**

```shell
    step {ALIAS} {COMMAND}
```

**Store**

```shell
step store customer_1 -c "ssh user@IP"
```

**init**

```
step init
```

**load**

```
step load
```

**Example Call**

```shell
    customer_1
```

**Need a hint ?**

```shell
    step -h
```
